## 2. Riemann Hypothesis / 리만 가설

### Theorem RH-131: `FiniteDimensionalPositivityCannotCertifyUniversalWeilPositivity`

Let `F` be a proper finite-dimensional subspace of a real inner-product space.
There are continuous quadratic forms `Q0` and `Q1` such that

```text
Q0(x)=Q1(x)>=0 for every x in F,
Q1(u)<0 for some u orthogonal to F.
```

### Proof / 증명

Choose a unit vector `u` orthogonal to `F` and put

```text
Q0(x)=||x||^2,
Q1(x)=||x||^2-2|<x,u>|^2.
```

For `x in F`, `<x,u>=0`, so `Q1(x)=Q0(x)>=0`. But `Q1(u)=-1`.
Both forms are continuous. `QED`

한국어로 말하면, 유한 개 bump, 유한 Gram 행렬 또는 유한 Galerkin 절단에서
양성이 확인되어도 아직 보지 않은 직교 방향에 음수가 숨어 있는 연속 이차형식을
배제하지 못한다. 이 반례는 실제 Weil 형식의 반례가 아니라, **유한 양성 데이터만
사용하는 추론 방식**의 반례다.

TICKET-130의 반결정 절차는 여전히 유효하다. 실제 Weil 형식에 엄격한 음수
핵심 원소가 있으면 언젠가 찾는다. 그러나 RH가 참일 때 탐색이 종료되거나 전칭
비음성을 인증한다는 뜻은 아니다.

```text
Discard / 폐기:
finite Gram positivity => universal Weil positivity

Next / 다음:
WeilSpecificCoerciveTailOrMonotoneOperatorLimit
```

필요한 것은 유한 절단 밖의 꼬리를 실제 Weil 구조로 제어하는 정리다. 예를 들면
절단 오차가 양의 coercive lower bound보다 작다는 균일 부등식, 또는 극한에서
양성을 보존하는 단조 self-adjoint operator 수렴 정리가 필요하다.

## 3. Collatz Conjecture / 콜라츠 추측

Use the accelerated odd map

```text
x_(i+1)=(3x_i+1)/2^a_i,
a_i=v_2(3x_i+1),
S_j=a_0+...+a_(j-1).
```

For a finite valuation word, define `B_0=0` and

```text
B_(j+1)=3B_j+2^S_j.
```

Then

```text
2^S_j x_j = 3^j n + B_j.
```

Exact valuation through step `j` selects one canonical odd residue `r_j` modulo
`2^(S_j+1)`:

```text
r_j == (2^S_j-B_j)(3^j)^(-1) mod 2^(S_j+1).
```

### Theorem CO-131A: `NaturalRealizationIffCylinderResiduesEventuallyStabilize`

An infinite positive valuation word is generated by one fixed positive integer `n`
if and only if its nested canonical start residues `r_j` are eventually constant.
The final constant is `n`.

### Proof / 증명

The exact cylinders are nested and their moduli tend to infinity because every
`a_i>=1`. If a fixed positive integer `n` lies in every cylinder, then for all large
`j` the modulus exceeds `n`; the canonical representative is therefore exactly `n`.
Conversely, if the nested representatives are eventually the same positive integer
`n`, then `n` belongs to every later cylinder and, by nesting, to every earlier
cylinder. It realizes every exact valuation prefix. `QED`

이 정리는 TICKET-130이 요구했던 “2-adic 경로와 자연수 크기의 결합”을 정확한
필요충분조건으로 닫는다. 다만 모든 no-descent 코드에서 안정화를 배제하는 것은
여전히 전칭 문제다.

### Theorem CO-131B: `StrictCapAllTimePromotionNotEstablishedByEnvelope`

Let `n` be a hypothetical least counterexample and suppose the verified lower bound
is `n>=H=2^28`. Minimality gives the all-time necessary inequality

```text
2^S_j <= 3^j(1+1/(3n))^j
       <= 3^j((3H+1)/(3H))^j.
```

TICKET-129 proved the stricter consequence

```text
S_j <= ceil(j log_2 3)
```

only for `j<=2H=2^29`. It does not follow for every `j` from the displayed
all-time envelope.

### Exact scope certificate / 정확한 범위 인증

At `j=3H`, the binomial theorem gives

```text
(1+1/(3H))^(3H) = 1 + 1 + positive terms > 2.
```

Thus the argument that placed the multiplicative correction below `2` has certainly
expired by `3H=805,306,368`. This does not prove that a counterexample violates the
strict cap, and it does not prove that an all-time strict cap is false. It proves that
all-time strict-cap necessity does not follow from the envelope established by
TICKET-129.

따라서 TICKET-130이 제시한
`ArchimedeanNaturalExclusionForAllInfiniteCapPaths`는 현재 전제만으로는 충분성이
증명되지 않았다. 모든 무한 strict-cap 경로를 배제해도, 가상의 최소 반례가
`2^29` 이후에도 strict cap을 지킨다는 별도 정리가 없기 때문이다. TICKET-130의 cap 언어 비소멸
정리와 지수적 밀도 정리는 독립된 유한 언어 정리로 유지하지만, 다음 전칭
목표는 폐기한다.

```text
Discard / 폐기:
ArchimedeanNaturalExclusionForAllInfiniteCapPaths as a sufficient Collatz target

Next / 다음:
NoEventuallyStableNaturalPathUnderExactNoDescentEnvelope
```

기계 감사에서는 TICKET-130의 mechanical strict-cap word를 512단계까지 계산했다.
최종 cylinder modulus는 813 valuation bits를 가지며 canonical residue는 809 bits다.
모든 검사 checkpoint에서 residue가 바뀌었다. 이는 안정화가 512단계까지
관측되지 않았다는 유한 사실일 뿐, 영원히 안정화하지 않는다는 증명이 아니다.
별도로 `a_i in {1,2,3,4}`인 깊이 1부터 4까지의 정확한 cylinder word 340개를
canonical residue에서 실제 재생했고 valuation 불일치는 0개였다.

## 4. Strong Goldbach Conjecture / 강한 골드바흐 추측

TICKET-129 uses the sufficient endpoint inequality

```text
A - K/log(H) - contamination > 0,
H=4*10^18,
A>131917/100000,
log(H)>214/5.
```

For even `N`, the binary singular series contains the arithmetic factor

```text
product_(p|N,p>2) (p-1)/(p-2).
```

### Theorem GB-131: `ArithmeticStratifiedGoldbachResidualBudgets`

If `N` has an odd prime divisor `p<=103`, then under the same fixed endpoint
constants a classwise pointwise estimate

```text
|R(N)| <= 57N/log N
```

would suffice above `H`. The exact worst-case margin at `p=103` is

```text
98007974997 / 216140000000000 > 0.
```

At `p=107`, the same coarse rational certificate has margin

```text
-1982688763 / 44940000000000 < 0.
```

Because `(p-1)/(p-2)` decreases with `p`, 103 is the largest prime threshold
certified by these fixed constants. The negative value at 107 means only that this
certificate stops there; it is not a Goldbach counterexample and is not an
impossibility theorem for `K=57`.

| classwise residual `K` | largest certified odd divisor `p` | covered density among even integers | unsieved rough density |
|---:|---:|---:|---:|
| 57 | 103 | 0.764061 | 0.235939 |
| 58 | 37 | 0.702558 | 0.297442 |
| 59 | 23 | 0.672824 | 0.327176 |
| 60 | 17 | 0.638949 | 0.361051 |
| 61 | 13 | 0.616384 | 0.383616 |
| 62 | 11 | 0.584416 | 0.415584 |
| 63 | 7 | 0.542857 | 0.457143 |
| 64 | 7 | 0.542857 | 0.457143 |

밀도는 해당 작은 홀수 소수들로 나누어지지 않는 정수의 정확한 유한 Euler
곱을 소수 체로 환산한 값이다. 확률적 증명이 아니라, 분석 정리를 어떤
산술형부터 공격할지 정하는 작업 분할이다.

```text
Next / 다음:
PointwiseBinaryGoldbachResidualByRoughnessStratum
```

먼저 작은 홀수 소인수를 가진 약 76.4%의 산술형에서 더 약한 `K=57` 목표를
공격하고, 남은 103-rough 짝수와 2의 거듭제곱을 별도 minor-arc 문제로 분리한다.
어떤 층에서도 점별 잔차 정리 자체는 아직 증명되지 않았다.

## 5. Twin Prime Conjecture / 쌍둥이 소수 추측

TICKET-130 defined

```text
Q=A/K,
a=Delta A/A_Y,
k=Delta K/K_Y.
```

### Theorem TP-131: `RelativeIncrementTargetIsExactReparameterization`

The dimensionless imbalance is exactly

```text
(a-k)/(1+k) = Q_X/Q_Y - 1.
```

Therefore

```text
R(Y)<2/23
```

is exactly the within-block ratio ceiling

```text
Q_X/Q_Y < 25/23.
```

At the endpoint candidate `Q_Y=23/25`, the threshold is sharp because

```text
(23/25)(25/23)=1.
```

### Interpretation / 해석

The identity is correct, but without an additional restriction derived from the
actual Vaughan coefficients it is only a reparameterization of the desired all-scale
bound. It does not make the analytic theorem easier and it supplies no parity
information.

즉, TICKET-130의 식은 잘못된 식이 아니지만 이를 “새로운 더 작은 핵심 난제”로
표현한 것은 과했다. 실제 진전이 되려면 다음 원시 계수 항등식의 분자에 대해
부호를 보존한 균일 상계를 증명해야 한다.

```text
Q_X-Q_Y = (Delta A-Q_Y Delta K)/K_X.
```

```text
Discard / 폐기:
R<2/23 alone as an analytically simpler target

Next / 다음:
UniformSignedVaughanBlockTransportWithParityBridge
```

이 목표도 두 부분으로 분리해야 한다. 첫째는 실제 Möbius/divisor 계수에서 오는
signed block cancellation이다. 둘째는 그 가중 하계가 정확한 간격 2의 소수쌍
무한성으로 넘어가도록 하는 parity-sensitive bridge다.

## 6. 증명 가능성 평가 / Proof viability assessment

### 가장 가까운 것

“논문형 조건부 정리”에 가장 가까운 것은 Goldbach다. finite glue의 모든 상수와
오염항이 닫혀 있고, 이번 산술형 층화로 분석 목표를 분할했다. 그러나 남은
pointwise binary minor-arc 추정은 바로 고전적 난점이며, 작은 상수 개선만으로
자동 해결되지 않는다.

“정확한 계산-논리 인터페이스”가 가장 선명한 것은 Collatz다. 무한 valuation
코드가 자연수 하나에서 발생하는 조건을 residue stabilization으로 정확히
표현했다. 하지만 모든 no-descent 안정 경로를 배제하는 순간 원래 추측의
보편성을 다시 마주한다.

RH는 반례 탐색이 계산 가능하다는 한쪽 방향만 닫혔다. Twin Prime은 실제
Vaughan 데이터가 풍부하지만 coefficient transport와 parity라는 서로 다른
무한 장벽이 동시에 남아 있다.

### 정직한 결론

현재 저장소는 네 문제를 해결할 “충분히 많은 도구”를 가진 상태가 아니다.
대신 유한 계산을 증명으로 오인하는 경로, 불충분한 보간, 범위를 벗어난 cap
확장, 재매개화를 난점 축소로 부르는 오류를 탐지하는 강한 연구 장치를 갖췄다.
다음 진전의 기준은 계산 범위 확대가 아니라 다음 중 하나여야 한다.

1. 모든 규모에 적용되는 새 부등식과 명시적 양의 여유;
2. 실제 반례를 주는 엄격한 interval certificate;
3. 기존 목표가 충분하지 않음을 보이는 정확한 countermodel;
4. Lean/Coq 수준으로 형식화 가능한 무한 귀납 또는 operator-limit 정리.
