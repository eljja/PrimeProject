# Collatz Fixed-Prefix Boundary Orbit

## Claim boundary / 주장 경계

This note classifies one exact 2-adic boundary orbit of the accelerated Collatz map. It does not prove the Collatz conjecture and it does not construct a positive-integer counterexample.

이 문서는 가속 Collatz 사상의 한 정확한 2-adic 경계 궤도를 분류한다. Collatz 추측을 증명하지 않으며 양의 정수 반례도 만들지 않는다.

## Correction record / 교정 기록

The first TICKET77 draft made an invalid inference: it treated an odd successor quotient as extinction of the stable prefix. An odd quotient means that the next valuation reaches the new modulus boundary exactly. TICKET76 already established that an equality-reaching valuation is not lift-stable and must be rolled back before another refinement. Therefore odd parity terminates only a **strictly-beyond-boundary pressure segment**. It does not terminate the normalized stable-prefix orbit.

첫 TICKET77 초안은 홀수 successor quotient를 안정 접두사의 소멸로 잘못 해석했다. 홀수 quotient는 다음 valuation이 새 modulus 경계에 정확히 닿는다는 뜻이다. TICKET76에 따르면 equality step은 lift-stable하지 않으므로 다음 refinement 전에 rollback해야 한다. 따라서 홀수 parity는 **경계를 엄격히 넘는 pressure 구간**만 끝내며, 정규화된 안정 접두사 궤도는 끝내지 않는다.

The invalid extinction claim and its artifacts were removed. The corrected result follows.

## Definitions / 정의

For odd positive integers, let

```text
T(n) = (3n+1) / 2^v2(3n+1).
```

Fix a lift-stable valuation prefix of length `m`, consuming `s` bits in a residue cylinder modulo `2^b`. Put

```text
d = b-s,
A = (3*T^m(r)+1)/2^d,
u = 3^(m+1).
```

Let `h(A)` be the unique digit in `{0,...,15}` satisfying

```text
A+h(A)*u = 0 mod 16,
```

and define

```text
P(A) = (A+h(A)*u)/16.
```

Because `u` is odd, `h(A)` exists and is unique. TICKET76 gives

```text
v2(A+h(A)*u) = 4+v2(P(A)).
```

If `P(A)` is even, the valuation lies strictly beyond the new four-bit boundary. If `P(A)` is odd, it reaches that boundary exactly and is rolled back by stable-prefix normalization. In both cases the next normalized boundary quotient is `P(A)`.

## Theorem / 정리

**FixedStablePrefixBoundaryOrbitAndTwoAdicGhostClassification.**

For every reachable fixed stable prefix:

1. the normalized boundary quotient eventually enters `1<=A<3^(m+1)`;
2. inside that interval it follows the complete `16^(-1)` orbit in one nonzero residue class modulo three;
3. that orbit has exact period `3^m`;
4. its compatible all-depth cylinders converge to a 2-adic point `N` satisfying `T^m(N)=-1/3`;
5. the limit is not a positive integer and is not a Collatz counterexample.

모든 도달 가능한 고정 안정 접두사에서 정규화된 경계 몫은 결국 유한한 `16` 역원 궤도에 들어가며 주기는 정확히 `3^m`이다. all-depth compatible cylinder의 극한은 `T^m(N)=-1/3`인 2-adic ghost이고 양의 정수가 아니다.

## Proof / 증명

### Lemma 1: reachability and contraction

The boundary identity is

```text
2^d*A = 3*T^m(r)+1.
```

Hence `3` does not divide `A`. Also

```text
16*P(A)=A+h(A)*3^(m+1),
```

so `P(A)=A mod 3` and nondivisibility by three is preserved. If `A>u`, then

```text
0<P(A)<=(A+15u)/16<A.
```

The orbit therefore enters `A<=u`; it cannot equal `u` because `3` does not divide `A`. Thus it enters `1<=A<u`.

경계 항등식에서 `3∤A`이고 이 조건은 `P` 아래 보존된다. `A>u`에서는 엄격히 감소하므로 결국 `1<=A<u`에 들어간다.

### Lemma 2: exact modular orbit

For `1<=A<u`, the same bounds give `0<P(A)<u`, and

```text
P(A) = 16^(-1)*A mod u.
```

The lifting-the-exponent lemma gives

```text
v3(16^(3^j)-1)=j+1,
```

hence

```text
ord_(3^(m+1))(16)=3^m.
```

The subgroup generated by `16` is the full unit class congruent to `1 mod 3`. Multiplying by the initial `A` shows that its orbit is exactly every unit below `u` with the same residue modulo three as `A`. The orbit period is therefore `3^m`.

`16`이 생성하는 부분군은 `mod 3`에서 1인 unit class 전체이며, `A`의 궤도는 `A mod 3`가 같은 unit 전체를 순회한다.

### Lemma 3: parity classifies strict segments, not stable-prefix extinction

For `m>=1`, each nonzero residue class modulo three below `u` contains even and odd representatives: `1,4` for class one and `2,5` for class two. Hence every periodic orbit encounters odd `P(A)`. This proves that a run of successors whose valuation lies **strictly** beyond each successive four-bit boundary is finite.

It does not prove stable-prefix extinction. At odd `P(A)`, the valuation consumes exactly the available bits. The equality step is rolled back, leaving the same stable prefix and quotient `P(A)` for the next refinement. The normalized orbit continues around the full modular cycle.

따라서 엄격한 pressure 구간은 끝나지만 equality rollback 뒤 동일한 안정 접두사 경계 궤도는 계속된다. 이 구분이 첫 초안의 오류를 고친 핵심이다.

### Lemma 4: the all-depth completion is a 2-adic ghost

Choose the canonical digit `h(A)` at every four-bit refinement. The compatible residue cylinders define a limit `N` in `Z_2`. At refinement depth `k`, the first valuation after the stable prefix is postponed beyond an arbitrarily high boundary. Therefore

```text
3*T^m(N)+1 = 0 in Z_2,
T^m(N) = -1/3.
```

For a positive integer `n`, every accelerated iterate `T^m(n)` is a positive odd integer, so it cannot equal `-1/3`. The fixed-prefix all-depth completion is not an admissible natural-number counterexample.

canonical digit를 계속 선택한 cylinder의 2-adic 극한은 `T^m(N)=-1/3`을 만족한다. 양의 정수의 가속 iterate는 양의 홀수이므로 이 극한은 자연수가 아니다.

The four lemmas prove the theorem. `QED`.

## Remaining infinite bridge / 남은 무한 연결고리

The fixed-prefix compatible limit is non-natural, but a positive integer may shadow a succession of different finite ghost prefixes. PrimeProject has no theorem preventing infinitely many changing-prefix approximation events.
